# Aleksandrs Drozdovs - Portfolio

Personal portfolio website built with Next.js 16, Tailwind CSS v4, and Framer Motion.

Live at: [aleksandrs-portfolio.vercel.app](https://aleksandrs-portfolio.vercel.app)

## Tech Stack

- **Framework:** Next.js 16 (App Router, SSG)
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Language:** TypeScript
- **Fonts:** Geist (headings), Inter (body)
- **Deployment:** Vercel

## Sections

- Hero with animated rotating subtitle
- About
- Skills and tech stack
- Projects (FYP Trading Strategy, Gym Membership System, Login/Signup UI)
- Beyond the Code (Judo, Algo Trading, Fashion)
- Contact

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Running Tests

```bash
npm test
```
